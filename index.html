Bandit

bandit es un juego para principiantes en el cual aprenderemos las bases para navegar y mirar el contenido de archivos en un s.o GNU/Linux , esto nos da las bases para seguir con los otros juegos; este juego consta de 34 niveles, para cada nivel hay unas indicaciones de lo que debemos lograr, para acceder a cada nivel se hace por medio de ssh

ssh credenciales

host: bandit.labs.overthewire.org

port: 2220

tener presente que para conectarnos necesitamos un user y una contrasena, estos van cambiando según el nivel, en el nivel 0 el user es bandit0 y la contrasena es bandit0 pero esto solo no lo dan en este nivel, porque la idea del juego es que encontremos la contrasena del siguiente nivel que está en un archivo en alguna parte del FHS, teniendo que usar los comandos necesarios para encontrarlo y leer su contenido.

a continuación se muestran los comandos que se necesitan para resolver el juego, solo se muestra una parte básica ya que cada comando tiene mas opciones y algunos como ssh permiten configuraciones en ~/.ssh/config

ssh

secure shell, protocolo de red que permite a los usuarios conectarse de forma segura a otro sistema a través de una red insegura, desarrollado en 1995 para reemplazar tecnologias como telnet, rlogin, ftp que transmiten datos sin cifrar.


ssh-1 1995 fue la primera versión.

ssh-2 1999 es el estandar actual ya que ofrece mejores caracteristicas de seguridad.


se usa principalmente para el acceso remoto a servidores, transferencia de archivos, ejecución remota de comandos y túneles seguros.

el software mas usado en ssh es openssh que es código abierto, en algunas distribuciones linux y mac vienen instaladas.

para que funcione ssh debe de haber un host que actue como servidor (corre el servcio sshd) y el cliente usa ssh, luego viene la autenticación donde se pide un user y una contrasena, para esto es mejor usar ssh keys que es mas seguro, todo esta comunicación es cifrada usando AES


el uso básico de ssh es el siguiente

user@domain

ssh bandit0@bandit.labs.overthewire.org

tambien se puede especificar el puerto, aunque por defecto es el 22, se recomienda cambiarlo, para especificarlo usamos la opción -p

ssh -p 2220 bandit0@bandit.labs.overthewire.org

luego nos pide ingresar la contrasena, despues vemos que nos manda una alerta para que indiquemos si confiamos en este servidor y debemos decir yes

también está la opción de usar una clave rsa, para esto usamos la opción -i
ssh -p 2220 -i rsakey bandit0@bandit.labs.overthewire.org

base64

herramienta que nos ayuda decodificar o codificar texto o archivos usando el esquema de base64, útil cuando converitmos de binario a texto legible y viceversa, por ejemplo una imagen convertimos a texto, y luego este texto lo convertimos de nuevo a imagen
tener presente lo siguiente, la codificación Base64 aumenta el tamaño del archivo en un 33% mas o menos y no es un método de encriptación seguro
base64 [opciones] [archivo...]

file | <<< text
podemos codificar todo un archivo o texto que indiquemos a base64 esto sucede porque cada caracter se convierte a su valor binario, sea que se use ascii o utf-8 cuando ya se tiene el valor binario de cada uno se agrupan en bloques de 24bits (3 bytes), luego estos bloques se dividen en 4 grupos de 6 bits, luego estos 6 bits se convierte a un número dentro 0 y 63, se mira la tabla ascii para saber a que corresponde ( esta tabla va desde la A hasta la Z, luego desde la a hasta la z, luego los número dentro del 0 y el 9, y los caracteres + y /), y para decodificar se hace todo este proceso a la inversa, si los datos originales no son un múltiplo exacto de 24 bits (por ejemplo, si solo hay 1 o 2 bytes restantes en el último bloque), Base64 agrega = como caracteres de relleno para completar la codificación.
1. convertir a binario
M = 01001101
a = 01100001
n = 01101110
2. agrupar en 24 bits(3 bytes)
01001101 01100001 01101110
3. agrupar en 4 grupos de 6 bits
010011 010110 000101 101110
4. convertir estos bits a decimal
19 22 5 46
5, mirar en la tabla de base64 a que corresponde cada número
19 = T, 22 = W, 5 = F, 46 = u
Man → TWFu
